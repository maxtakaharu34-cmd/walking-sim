# Walking Sim — Open Skies

A quiet 3D walking-simulator vignette built with Three.js (vanilla, no build step).

## Play online

Enable GitHub Pages on this repo (`Settings → Pages → Source: main / root`) then visit:

`https://<your-username>.github.io/walking-sim/`

## Controls

- **Click canvas** — start
- **WASD / Arrow keys** — move
- **Mouse** — look
- **Shift** — sprint
- **Space** — jump
- **Mouse wheel** — zoom
- **Esc** — release pointer

## Tweaks

Tap the floating **Tweaks** panel to change time of day, wind, fog, render quality, language, and toggle the demo (auto-orbit) camera.

## Files

- `index.html` — entry
- `scene-aaa.jsx` — world / terrain / lighting / wildlife
- `tweaks-panel.jsx` — in-page tweaks UI

No bundler — open `index.html` directly or serve the folder with any static server.
